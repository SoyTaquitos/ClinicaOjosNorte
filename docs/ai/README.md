# AI MEMORY SYSTEM (docs/ai/)

## ¿Para qué sirve esta carpeta?
Este directorio actúa como la **Memoria Persistente Multi-Agente** del proyecto **Oftalmología SI1 — Clínica de Ojos Norte**.
Es el puente cognitivo para administrar y desarrollar el Sistema de Información de Gestión (Web + Backend). No hay app móvil.

## Instrucciones y Orden de Lectura para Agentes de Desarrollo:
1. **`MASTER_AGENT_PROMPT.md`** - Lee las reglas inquebrantables del proyecto. **Importante: los pacientes NO son usuarios del sistema.**
2. **`PROJECT_VISION.md`** - Internaliza qué sistema y problema estamos abordando.
3. **`ARCHITECTURE.md`** - Domina las directrices de arquitectura Web + Backend.
4. **`TECH_STACK.md`** - El arsenal tecnológico en sus versiones aprobadas.
5. **`CURRENT_STATE.md`** y **`HANDOFF_LATEST.md`** - Actualízate con lo que se venía tocando para evitar redundancias o sobrescribir código.
6. **`NEXT_STEPS.md`** - Escoge o suma metas.

**Jamás cierres o indiques al operador humano "TAREA FINALIZADA" sin actualizar esta carpeta con el estado actual.**
